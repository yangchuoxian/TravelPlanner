  REST API list

Web SPA:

post /submit_login -- Submit login with username and password
get /logout -- Log user out, logged in user id will be removed from server session
get /get_user_roles -- Get all user roles
get /get_users -- Get paginated users
get /search_users -- Get paginated search results of users, user's username or name will be searched to see if it contains search keyword
get /order_users -- Get paginated ordered users, users can be ordered by createdAt or updatedAt/lastTimeLogin
get /get_users_with_role -- Get paginated users with specific role
get /get_user_info -- Get single user information
post /update_user_role -- Update one or more users' role
post /update_basic_user_info -- update single user information
post /upload_user_avatar -- Upload user avatar image file, the user can either be an existing user or a new user
get /user_avatar -- Get user avatar with given user id
post /submit_new_user -- Create a new user


get /get_schedules -- Get paginated schedules
get /get_schedule_info -- Get single schedule information with given schedule id
get /search_schedules -- Get paginated search results of schedules, schedule's cityOfDeparture, cityOfArrival, comment or username will be searched to check if any of it contains the search keyword
get /order_schedules -- Get paginated ordered schedules, schedules can be ordered by startDate
post /delete_schedule -- Delete one or more schedules
post /submit_new_schedule -- Create a new schedule
post /update_basic_schedule_info -- Update schedule

iOS:

post /mobile/submit_login -- Submit login from iOS app with username and password, after login, a loginToken will be generated to help identify further requests from iOS to check whether this user is authenticated or not
post /mobile/update_user -- Update user information from iOS app
get /mobile/logout -- Log out the user from iOS app, the loginToken for this user saved in database will be removed
post /mobile/change_user_password -- Change user password, old password needs to be provided
post /mobile/upload_user_avatar -- Upload user avatar for either an existing user or a new user that is registering at the moment
post /mobile/submit_new_user -- User registration, username, password and email needed
get /mobile/user_avatar -- Get and download user avatar image
get /mobile/get_user_info -- Get user information


get /mobile/filter_schedules -- Get paginated schedules whose startDates are within a given date range
get /mobile/search_schedules -- Get paginated search results of schedules, schedule's cityOfDeparture, cityOfArrival, comment or username will be searched to check if any of it contains the search keyword
post /mobile/change_schedule -- Change a single schedule information
post /mobile/delete_user_schedule -- Delete a schedule that currently logged in user owns
post /mobile/create_new_schedule -- Create a new schedule for currently logged in user
